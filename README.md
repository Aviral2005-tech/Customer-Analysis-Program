# 📊 Customer Sales Analysis Dashboard

A comprehensive data analytics project that analyzes customer purchasing behavior, sales performance, and churn trends using **Python**, **Pandas**, **NumPy**, **Matplotlib**, and **Seaborn**. The project performs data cleaning, exploratory data analysis (EDA), customer segmentation, sales aggregation, pivot table analysis, and visualization to generate actionable business insights.

---

## 📌 Project Overview

This project demonstrates an end-to-end sales analytics workflow by analyzing historical sales transactions and customer churn data. It provides valuable insights into customer purchasing patterns, top-performing products, regional sales distribution, monthly revenue trends, and customer retention.

The project also generates a professional dashboard and exports analytical reports for further business decision-making.
